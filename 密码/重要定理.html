互质关系

如果两个正整数，除了1以外，没有其他公因子，我们就称这两个数是[互质关系](https://zh.wikipedia.org/zh-cn/互素)（coprime）。比如，15和32没有公因子，所以它们是互质关系。这说明，不是质数也可以构成互质关

最大公约数

a、b为整数
gcd(a,b) = gcd(b,a mod b) 欧几里得算法
gcd(a,b) = ax + by 扩展欧几里得算法 (x,y为整数)
gcd(a,b) = gcd(a+-b, b)
gcd(na,nb) = ngcd(a,b)
gcd(a,b)= ab/lcm(a,b)

欧拉定理

a、n互质
a^φ ( n ) ≡ 1 mod n

欧拉函数

欧拉函数是小于N且与N互质的数的个数n、p1,p2...pn为素因子。
φ(n) = n*(1-1/p1)*(1-1/p2)*......(1-1/pn) 

情况1
如果n=1，则 φ(1) = 1 。因为1与任何数（包括自身）都构成互质关系。

情况2
如果n是质数，则 φ(n)=n-1 。因为质数与小于它的每一个数，都构成互质关系。比如5与1、2、3、4都构成互质关系。

费马小定理

n为质数，a、n互质则：
a^(n-1) ≡ 1 mod n
或 a^n ≡ a mod n
