# 汇编指令



| 指令格式 | 说明                                        | 备注                                |
| -------- | ------------------------------------------- | ----------------------------------- |
| lodsb    | 读取esi指向的地址一个字节内容，并赋值到al中 | 方向标志D=0，esi递增；D=1，esi递减  |
| stosb    | 将al内容赋值到edi地址指向的空间中           | 方向标志D=0，edii递增；D=1，edi递减 |
